#  模組介紹：


模組的作用是管理 檔案內程式碼的權限 （類似Spring Boot 一個命名空間有誰能用）

import {
  要將其他模組內的程式碼導入近來要用：      imports [...],
 
  自身模組如果是一個 路由 要用掛載同層級的  controller:[...],
  
  自身模組內的程式碼 要共用共享要用：       providers: [...],

  要將一個模組內的程式碼共用出去要用：      exports: [...],


}
基本上 providers 跟 exports 會一樣 


# services.ts  是一個類 裡面有方法 要將方法放到DI 要加入 @Injectable()
然後要 "使用" 這個類 要建立 "類實例" (語法依照使用方式會不一)

# controller 裡面的"裝飾符" 是"路由" 的意思 要用 @Controller('路由名稱')
